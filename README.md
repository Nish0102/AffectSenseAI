# 🎭 AffectSense-AI  
### Real-Time Facial Emotion Recognition System using Deep Learning  

---

## 🚀 Overview  
AffectSense-AI is a real-time facial emotion recognition system that detects human emotions through webcam input using deep learning and computer vision.  

The system uses a fine-tuned ResNet18 model trained on the FER2013 dataset to classify emotions such as:  
**Happy, Sad, Angry, Neutral, Surprise, Fear, and Disgust**  

It also predicts additional attributes like **intensity** and **valence**.

---

## ✨ Features  
- 🎥 Real-time webcam emotion detection  
- 🧠 Deep learning model using PyTorch  
- ⚡ Transfer learning (ResNet18) for higher accuracy  
- 🎯 Confidence filtering to reduce wrong predictions  
- 🔁 Prediction smoothing for stable output  
- 😊 Multi-head outputs:
  - Emotion classification  
  - Intensity prediction  
  - Valence prediction  
- 📊 Training visualization (accuracy & loss graphs)
